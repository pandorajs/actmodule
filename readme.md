#listview


 > 内容中心通用活动组件

##how to demo

1. checkout
1. run `npm install`
1. run `spm install`
1. run `grunt`
1. view files in `/demo`

##how to use

1. run `spm install pandora/actmodule`
1. write `require('pandora/actmodule/VERSION.NUMBER/actmodule')`

##find examples

1. view the source files in '/src'
